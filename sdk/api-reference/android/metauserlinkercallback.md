# MetaUserLinkerCallback

This interface is used with [`MetaUserLinkerAsync`](metauserlinkerasync.md#link).

## MetaUserLinkerCallback API

### `onSuccess()`

> ```java
> void onSuccess()
> ```
>
> Call this method after linking succeeds.

### `onFailure()`

> ```java
> void onFailure()
> ```
>
> Call this method after linking fails.
