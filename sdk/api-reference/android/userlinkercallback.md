# UserLinkerCallback

This interface is used with [`UserLinkerAsync`](userlinkerasync.md).

## UserLinkerCallback API

### `onSuccess()`

> ```java
> void onSuccess()
> ```
>
> Call this method after linking succeeds.

### `onFailure()`

> ```java
> void onFailure()
> ```
>
> Call this method after linking fails.
