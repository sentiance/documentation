## HarshDrivingEvent API

### `getTime()`

> ```java
> DateTime getTime()
> ```
>
> Returns the date and time of occurrence of the event.

### `getHarshDrivingEvents()`

> ```java
> double getMagnitude()
> ```
>
> Returns the magnitude of the harsh event, in m/s^2.
