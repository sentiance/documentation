# GeoLocation

Represents a geographic location.

## GeoLocation API

|  |  |
| :--- | :--- |
| double | getLatitude \(\) |
| double | getLongitude \(\) |
| int | getAccuracyInMeters \(\) |



### `getLatitude()`

> ```java
> double getLatitude()
> ```
>
> Returns the latitude in degrees.

### `getLongitude()`

> ```java
> double getLongitude()
> ```
>
> Returns the longitude in degrees.

### `getAccuracyInMeters()`

> ```java
> int getAccuracyInMeters()
> ```
>
> Returns the accuracy in meters.
