# Attribute

Represents an attribute pertaining to a moment or a segment.

## Attribute API

|  |  |
| :--- | :--- |
| String | [getName](attribute.md#getname) \(\) |
| Float | [getDeltaV](attribute.md#getvalue) \(\) |



### `getName()`

> ```java
> String getName()
> ```
>
> Returns the name of the attribute.

### `getValue()`

> ```java
> double getValue()
> ```
>
> Returns the value of the attribute.
