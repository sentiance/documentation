# PoiAddress

Represents a reverse geocoded location.

## PoiAddress API

|  |  |
| :--- | :--- |
| String | [getCity](poiaddress.md#getcity) \(\) |
| String  | [getCityType](poiaddress.md#getcitytype) \(\) |
| String | [getCountry](poiaddress.md#getcountry) \(\) |



### `getCity()`

> ```java
> String getCity()
> ```
>
> Returns the city.

### `getCityType()`

> ```java
> String getCityType()
> ```
>
> Returns the city type.

### `getCountry()`

> ```java
> String getCountry()
> ```
>
> Returns the country.
