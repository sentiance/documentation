# PoiAddress

Represents a reverse geocoded location.

## PoiAddress API

### `getCity()`

> ```java
> String getCity()
> ```
>
> Returns the city.

### `getCityType()`

> ```java
> String getCityType()
> ```
>
> Returns the city type.

### `getCountry()`

> ```java
> String getCountry()
> ```
>
> Returns the country.
